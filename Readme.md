# セミドライスーツ配色シミュレーター

スマホで使える、セミドライスーツの色決めアプリです。  
部位ごとに色を選び、ステッチ色を変更し、最終的に画像や仕様書PNGとして保存できます。

## はじめて使う方へ（3分ガイド）
1. 画面左のスーツ画像をタップして、色を変えたい部位を選びます。  
2. 出てきたカラーパレットから色を1つ選びます。  
3. 右上の`ステッチ`から、縫い目の色を選びます。  
4. 必要なら`おすすめ生成`や`プリセット`で全体色を調整します。  
5. 最後に`スーツ画像保存`または`仕様書PNG`で出力します。

## できること
- 部位タップで色変更
- ステッチ色変更
- 柄色対応（チェッカー赤/黒、カモカーキ、カモグレイ）
- プリセット適用
- テーマ別おすすめ生成（ランダム）
- 全身ブラックにリセット
- スーツ画像PNG保存
- 仕様書PNG保存 / 印刷 / 共有URLコピー

## 操作のコツ（スマホ）
- ピンチで拡大/縮小
- 1本指ドラッグで位置調整
- ダブルタップで等倍に戻す
- 色パレットやステッチパレットが開いている間は、背景の誤タップを抑制しています

## 固定仕様（選択不可）
以下の部位はブラック固定です。
- 首周り
- 裾
- リブ

## 注意点
- ロゴ編集機能は現在一時停止中です（追加・削除・移動はできません）。
- 膝パッドの選択色は制限されています（白/黒）。

## ローカルで起動する方法（開発者向け）
```bash
npm install
npm run dev
```

ブラウザで `http://localhost:5173` を開きます。

## ビルド / テスト
```bash
npm run build
npm run test
```

## 設定ファイル（色やプリセットを変更したい場合）
- `public/config/colors.json`: パネルカラー（`patternTile` で柄指定）
- `public/config/stitch-colors.json`: ステッチカラー
- `public/config/parts.json`: 部位定義（`allowColors` で部位ごとの選択制限）
- `public/config/presets.json`: プリセット
- `public/config/color-themes.json`: おすすめ生成テーマ
